# virtual-wallet
Virutal Wallet is developed using HTML,CSS and PHP .Virtual Wallet is used for showing the Payment through wallet and to add the balance into the wallet using virtual credit and debit card. The UserName or UserID will be taken from your project's session and based on it the account will be created in the database. Database contains UserName/UserID and total balance. And another table contains the invoice that contains the history about adding balance or purchasing from wallet.
